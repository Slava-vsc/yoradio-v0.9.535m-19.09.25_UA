опис Maleksm:
МОД на основе версии автора 0.9.533 с исправленными ошибками в Web-интерфейсе и переключателем SmartStart.
- Исходники audioI2S от Wolle (schreibfaul1) 3.4.2p от 13.09.2025.
- Добавлен дисплей ST7789 2,25' с разрешением 284x76. Добавлена поддержка I2S CS4344 и других ЦАП-ов с сигналом MCLK.
- Изменён Web-интерфейс, добавлен раздел поиска радиостанций по странам и добавления в плейлист.
- Изменения в батарейке, касающиеся методики расчётов и определения подключения зарядки.

Новый МОД работает начиная с v.3.0.4 по v.3.0.7, v.3.2.0, v.3.2.1 и выше. Рекомендую ядра v.3.0.7 или v.3.2.1.
Обновлённый MOD ё-Радио с новыми форматами для I2S и vs1053b.
Теперь играют форматы M3U8 и OGG, кодеки VORBIS, на I2S играют OPUS и FLAC.
Включает MOD с батарейкой для ВСЕХ популярных дисплеев, за исключением текстовых: LCD1602 и LCD2004.
Обновление. Добавлена возможность воспроизведения новых форматов. Исправлены известные ошибки распознавания потоков. Заменены аудио-библиотеки. 
На I2S новый VU-метр. 
Сеть режима точки доступа изменена на открытую. Вставлена функция плавного гашения подсветки с управлением из файла конфигурации. Добавлены конфиги для дисплеев с высоким разрешением.
МОД работает на ESP32-S3. Для компиляции и прошивки в Arduino IDE (v.1.8.19) можно использовать новые версии ядра (выше 3.0.x).
8. На I2S изменён вывод звука: сделан расемплинг звука до 48КГц.
9. В плеере флешек оставлено воспроизведение только файлов MP3 и FLAC с отображением метаданных.
10. На I2S полностью изменён VU-метр, теперь "прыгает" более адекватно. Введён алгоритм АРУ. На VS1053 он тоже чуть подправлен.
11. Работа ESP32 переведена на два ядра (!). Теперь ничего не тормозит, работают дисплеи с высоким разрешением (проверено до 480х320). Ядро 0 - основные задачи, сеть, периферия. Ядро 1 - работа декодеров, аудио-фильтры и воспроизведение.
